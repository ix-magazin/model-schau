Das Python-Programm `for i in range(100): print(i)` zählt die Zahlen von 0 bis 99, also **100 Zahlen**.

Hier ist die detaillierte Aufteilung:

*   **0:** `range(100)` beginnt beim ersten Element. Da der Startwert 0 ist, wird zuerst **0** printet.
*   **1 bis 98:** Es wird dann bis zum 98. Element printet. Das sind **98** Zahlen.
*   **99 bis 99:** Das letzte Element ist 99. Es wird dann **99** printet.
*   **100:** Das letzte Element ist 100. Da die Funktion `range(100)` endet (und nicht mehr läuft), wird **100** *nicht* printet.

**Zusammenfassung:**
Die Funktion zählt die Werte, die die Funktion `range(100)` für die Indexwerte `i` zurückgibt.
Mathematisch ist `range(start, stop)` eine Funktion, die eine Liste von **`stop - start`** Zahlen zurückgibt.
*   `stop` = 100
*   `start` = 0

Also: `100 - 0 = 100` Zahlen.

Der Programmcode zählt also **100** Zahlen.
