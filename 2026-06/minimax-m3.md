## 📌 Die Regel von `range()`

Python's `range()` arbeitet nach dem **„Halb-offen-Intervall"-Prinzip**:

- `range(stop)` → beginnt bei `0`, endet **vor** `stop`
- `range(start, stop)` → beginnt bei `start`, endet **vor** `stop`
- `range(start, stop, step)` → wie oben, mit Schrittweite `step`

**Der Stop-Wert ist immer exklusiv** (also nicht enthalten)!

## 1️⃣ `pivot` – Einfaches Umformen

**`pivot`** formt einen DataFrame von „langem" Format in „breites" Format um – **ohne Aggregation**.

```python
import pandas as pd

df = pd.DataFrame({
    'Datum':  ['2024-01-01', '2024-01-01', '2024-01-02', '2024-01-02'],
    'Stadt':  ['Berlin', 'Hamburg', 'Berlin', 'Hamburg'],
    'Wert':   [10, 20, 30, 40]
})

# Pivot: Zeilen = Datum, Spalten = Stadt, Werte = Wert
result = df.pivot(index='Datum', columns='Stadt', values='Wert')
print(result)
```

**Ausgabe:**

```
Stadt       Berlin  Hamburg
Datum
2024-01-01      10       20
2024-01-02      30       40
```

> ⚠️ **Problem:** Wenn es **Duplikate** gibt (z. B. zwei Zeilen mit gleichem Datum und gleicher Stadt), wirft `pivot` einen **`ValueError`**.

---

## 2️⃣ `pivot_table` – Umformen + Aggregieren

**`pivot_table`** ist die „mächtigere" Version von `pivot`. Sie kann **Duplikate aggregieren** (z. B. mit Summe, Mittelwert, Anzahl).

```python
df = pd.DataFrame({
    'Datum':  ['2024-01-01', '2024-01-01', '2024-01-01', '2024-01-02'],
    'Stadt':  ['Berlin', 'Berlin', 'Hamburg', 'Berlin'],
    'Wert':   [10, 15, 20, 30]   # ⚠️ 'Berlin/2024-01-01' kommt 2x vor
})

# Pivot-Tabelle mit Summen-Aggregation
result = df.pivot_table(
    index='Datum',
    columns='Stadt',
    values='Wert',
    aggfunc='sum'      # Default wäre 'mean'!
)
print(result)
```

**Ausgabe:**

```
Stadt       Berlin  Hamburg
Datum
2024-01-01      25       20   # 10+15 = 25
2024-01-02      30      NaN
```

### Wichtige Parameter von `pivot_table`

| Parameter | Bedeutung | Default |
|---|---|---|
| `aggfunc` | Aggregationsfunktion (`sum`, `mean`, `count`, `max`, ...) | `'mean'` |
| `fill_value` | Wert für fehlende Kombinationen | `None` |
| `margins` | Zeilen/Spalten-Gesamtsummen hinzufügen | `False` |
| `margins_name` | Name der Gesamtsumme | `'All'` |

```python
# Mit Gesamtsumme und NaN-Behandlung
df.pivot_table(
    index='Datum',
    columns='Stadt',
    values='Wert',
    aggfunc='sum',
    fill_value=0,
    margins=True,
    margins_name='Gesamt'
)
```

## 3️⃣ `crosstab` – Häufigkeitstabellen

**`crosstab`** ist speziell für **Kreuztabellen** (Contingency Tables) gedacht und wird meist für **Häufigkeitsanalysen** verwendet.

```python
import numpy as np

df = pd.DataFrame({
    'Geschlecht': ['m', 'w', 'm', 'w', 'm', 'w', 'm'],
    'Raucher':    ['ja', 'nein', 'ja', 'ja', 'nein', 'nein', 'ja'],
    'Alter':      [25, 30, 35, 40, 45, 50, 55]
})

# Häufigkeitstabelle: Geschlecht × Raucher
pd.crosstab(df['Geschlecht'], df['Raucher'])
```

**Ausgabe:**

```
Raucher     ja  nein
Geschlecht
m            3     1
w            1     2
```

### Wichtige Eigenschaften

- **Default-`aggfunc`** ist `'count'` (zählt die Häufigkeit)
- Akzeptiert **beliebige Arrays** (Series, Listen, NumPy-Arrays) – nicht zwingend einen DataFrame
- Unterstützt **Normalisierung** (z. B. relative Häufigkeiten)

```python
# Relative Häufigkeiten (in Prozent)
pd.crosstab(df['Geschlecht'], df['Raucher'], normalize='all') * 100
```

```python
# Mit Wert-Aggregation (z. B. Summe des Alters)
pd.crosstab(
    df['Geschlecht'],
    df['Raucher'],
    values=df['Alter'],
    aggfunc='mean'
)
```

## 🎯 Wann nutzt man was?

| Situation | Empfehlung |
|---|---|
| **Einfaches Umformen**, keine Duplikate | `pivot` |
| **Umformen mit Aggregation** (Summen, Mittelwerte etc.) | `pivot_table` |
| **Häufigkeiten zählen** (z. B. für Kontingenzanalysen) | `crosstab` |
| **Daten aus beliebigen Arrays** (nicht DataFrame) | `crosstab` |
| **Statistische Auswertungen** (Chi-Quadrat-Test etc.) | `crosstab` |

<summary>🔍 Zusammenfassung der Unterschiede</summary>

**`pivot`:**

- ❌ Wirft Fehler bei Duplikaten
- 🎯 Ideal für **bereits aggregierte Daten**

**`pivot_table`:**
- 📐 Reshape **+ Aggregation**

- 🛠️ Sehr flexibel (`margins`, `fill_value`, ...)
- 🎯 Die **„eierlegende Wollmilchsau"** für Tabellen

**`crosstab`:**
- 📊 Speziell für **Häufigkeiten**
- ✅ Arbeitet mit **beliebigen Arrays**
- 🎯 Perfekt für **statistische Analysen** und **Kreuztabellen**
