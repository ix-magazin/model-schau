# Was ist in pandas der Unterschied zwischen pivot und crosstab?



> The user asks in German about the difference between `pivot` and `crosstab` in pandas. Let me explain this well with code examples.
> 
> Key differences:
> 
> 1. **`pivot`** (DataFrame method / `pd.pivot`):
>    - Reshapes data: turns unique values of a column into new columns
>    - Requires unique index/column combinations (raises error on duplicates)
>    - No aggregation — just restructuring
>    - `pivot_table` is the aggregating version
> 
> 2. **`crosstab`** (`pd.crosstab`):
>    - Computes frequency tables (cross-tabulation) by default
>    - Works on Series/arrays directly, not necessarily a DataFrame
>    - Always aggregates (counts by default), can use `values` + `aggfunc`
>    - Handles duplicates naturally
>    - Has extra features like `normalize`, margins
> 
> So main distinction:
> - `pivot` = pure reshaping, no aggregation, fails on duplicate combinations
> - `crosstab` = frequency/aggregation table, defaults to counting occurrences
> - `pivot_table` sits in between: reshaping + aggregation
> 
> Examples:
> 
> ```python
> import pandas as pd
> 
> df = pd.DataFrame({
>     'Name': ['Anna', 'Ben', 'Anna', 'Ben'],
>     'Monat': ['Jan', 'Jan', 'Feb', 'Feb'],
>     'Umsatz': [100, 150, 120, 180]
> })
> ```
> 
> pivot with unique combos works:
> ```python
> df.pivot(index='Name', columns='Monat', values='Umsatz')
> ```
> 
> crosstab counts:
> ```python
> pd.crosstab(df['Name'], df['Monat'])  # frequency table
> ```
> 
> crosstab with values+aggfunc mimics pivot_table:
> ```python
> pd.crosstab(df['Name'], df['Monat'], values=df['Umsatz'], aggfunc='sum')
> ```
> 
> Duplicates: pivot raises `ValueError: Index contains duplicate entries, cannot reshape`; crosstab counts them.
> 
> Also: crosstab accepts arrays/lists directly without DataFrame; pivot is a DataFrame method (pd.pivot exists too but takes a DataFrame).
> 
> Formatting: German response, use table for comparison, code blocks with ``` fences, maybe headers. Moderate length. Should include a comparison table, code examples, and guidance on when to use which.


# Unterschied zwischen `pivot` und `crosstab` in pandas

Kurz gesagt: **`pivot` strukturiert nur um**, während **`crosstab` aggregiert** (standardmäßig zählt es Häufigkeiten).

## Vergleich im Überblick

| Aspekt | `pivot` | `crosstab` |
|---|---|---|
| **Zweck** | Reines Umformen (Reshaping) | Kreuztabelle / Häufigkeiten |
| **Aggregation** | ❌ Nein | ✅ Ja (Standard: Zählen) |
| **Duplikate** | Fehler bei doppelten Index/Spalten-Kombinationen | Kein Problem – werden gezählt/aggregiert |
| **Eingabe** | DataFrame | Beliebige Series/Arrays (auch ohne DataFrame) |
| **Extras** | – | `normalize`, `margins` (Summen) |

## Beispiel

```python
import pandas as pd

df = pd.DataFrame({
    'Name':   ['Anna', 'Ben', 'Anna', 'Ben'],
    'Monat':  ['Jan', 'Jan', 'Feb', 'Feb'],
    'Umsatz': [100, 150, 120, 180]
})
```

### `pivot` – nur umformen

```python
df.pivot(index='Name', columns='Monat', values='Umsatz')
```
```
Monat  Feb  Jan
Name
Anna   120  100
Ben    180  150
```

Die Werte werden nur **umverteilt** – es findet keine Berechnung statt. Gibt es für eine Kombination **mehrere Zeilen**, schlägt `pivot` fehl:

```
ValueError: Index contains duplicate entries, cannot reshape
```

### `crosstab` – zählt standardmäßig

```python
pd.crosstab(df['Name'], df['Monat'])
```
```
Monat  Feb  Jan
Name
Anna     1    1
Ben      1    1
```

Mit `values` und `aggfunc` wird daraus eine Aggregation:

```python
pd.crosstab(df['Name'], df['Monat'], values=df['Umsatz'], aggfunc='sum')
```

## Wann was verwenden?

- **`pivot`**: Jede Zeilen-/Spaltenkombination kommt **genau einmal** vor, du willst nur das Layout ändern.
- **`crosstab`**: Du willst **Häufigkeiten** sehen (z. B. „Wie oft kommt Kategorie X in Gruppe Y vor?") – auch schnell ohne DataFrame nutzbar:
  ```python
  pd.crosstab([1, 1, 2], ['a', 'a', 'b'])
  ```
- **`pivot_table`**: Der „dritte Weg" – wie `pivot`, aber **mit Aggregation** (z. B. Mittelwert bei Duplikaten). `crosstab` mit `values`+`aggfunc` ist praktisch ein Spezialfall davon.

**Faustregel:** Willst du nur umformen → `pivot`. Willst du zählen oder aggregieren → `crosstab` oder `pivot_table`.
